<h2>📌 Overview</h2>

<p>
MovieRecML is a Machine Learning-based movie recommendation system that suggests movies based on user preferences. It uses similarity techniques to provide accurate and fast recommendations along with movie posters.
</p>

<br/>

<h2>✨ Features</h2>

<ul>
  <li>🎯 Personalized movie recommendations</li>
  <li>⚡ Fast and responsive UI</li>
  <li>🖼️ Movie posters using TMDB API</li>
  <li>🧠 ML-based similarity model</li>
  <li>☁️ Deployed on Streamlit Cloud</li>
</ul>

<br/>

<h2>🧩 How It Works</h2>

<p><b>1. Select Movie</b><br/>
Choose a movie from the dropdown list.
</p>

<p><b>2. Recommendation Engine</b><br/>
System finds similar movies using cosine similarity.
</p>

<p><b>3. Results Display</b><br/>
Top 5 recommended movies are displayed with posters.
</p>

<br/>

<h2>🧠 ML Logic</h2>

<ul>
  <li>Text vectorization using CountVectorizer</li>
  <li>Cosine similarity for recommendation</li>
  <li>Precomputed similarity matrix</li>
</ul>

<br/>
